Realizacion de una pagina web con apache

Creado por: Carlos Mateos Samuel Cabrera y Ignacio De Los Riscos

Instalacion de Apache

La instalacion de apache la hemos realizado en una maquina virtual de debian y hemos creado una pagina web en ella.

Hacer accesible nuestra pagina desde fuera del area de trabajo

Hemos accedido a nuestro router como administrador y hemos abierto el puerto 80 y direccionado a la ip de nuestra maquina virtual. Esto hara que podamos acceder desde cualquier sitio a nuestra pagina.

Obtencion de DNS fijos

Hemos procedido a meternos en la pagina web de no ip y dar de alta nuesta ip publica para poder tener un enlace fijo para nuestra pagina de modo que se pueda acceder sin necesidad de usar la ip publica.
